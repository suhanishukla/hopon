## Cloning the Repo
1. cd Desktop
2. git clone <url link>
3. cd hopon

You are now in the hopon project folder!

## Setting up Dependencies
Client Side
1. cd client
2.  npm install --legacy-peer-deps


Server Side
1. cd server
2. npm install body-parser cors express mongoose nodemon
